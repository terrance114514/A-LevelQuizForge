# A-Level 试题锻造坊 (A-Level Quiz Forge)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**一款为 A-Level 学生设计的智能互动网页工具，用于分析错题并生成自定义模拟练习试卷。**

纯原生 JavaScript + HTML5 + CSS3 构建 —— 无框架、无后端，开箱即用。

🔗 **在线体验与更多信息**：[http://exhelper.cc/](http://exhelper.cc/)

---

## ✨ 功能特色

### 错题分析（演示版）
- 输入题目、学生答案、正确答案及可选的自评原因。
- 工具基于简单规则进行分析，并返回：
  - 错误类型（如计算精度、概念误解）
  - 相关知识点
  - 改进建议
  - 后续练习方向

### 模拟试卷生成器
- 根据所选年级、考试局、学科及试卷，可配置：
  - 题目数量（5–30）
  - 难度（基础 / 中等 / 挑战 / 混合）
  - 题型（选择题 / 结构化 / 混合）
  - 时长目标（30–120 分钟）
  - 知识点（多选）
- 使用本地题库（`examData.js`）按配置抽取题目。
- 生成的试卷和答案立即展示。

### 开箱即用
- 所有数据均为模拟数据；后续可替换分析规则和题库，接入真实 AI API 或数据库。

## 🚀 使用方法

1. 克隆或下载本仓库。
2. 用任意现代浏览器（Chrome、Edge、Firefox、Safari）打开 `index.html`。
3. 选择学习背景：
   - 年级（G1、G2、AS、A2）
   - 考试局（CIE、Edexcel、AQA）
   - 学科（数学、物理、化学）
   - 试卷（根据考试局和学科动态变化）
4. 选择功能：
   - **错题分析** – 填写表单，获得分析结果卡片。
   - **生成模拟试卷** – 配置参数，查看生成的题目和答案。
5. 所有选择自动保存在 `localStorage` 中，页面间切换不丢失。

## 🛠️ 技术栈

- **HTML5** – 语义化结构
- **CSS3** – 自定义属性、响应式网格/弹性布局、现代卡片设计
- **纯 JavaScript (ES5+)** – 无框架，轻量高效
- **LocalStorage** – 持久化用户选择
- **URLSearchParams** – 通过查询字符串在页面间传递配置
